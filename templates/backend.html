home.html - HTML file to establish Home page

index.html - HTML file to establish visualization page

Leaflet - Used to create map visualization

Chart.js - Used to create scatter plot and bar graph visualizations

Flask - Used to establish webpage and various routes

Jinja - Template engine for Flask; Used to link JavaScript, Python, and Database to help create webpage

SQLite- Used to create database
